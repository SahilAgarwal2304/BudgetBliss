<h1>BudgetBliss</h1>
<p>A simple personal finance tracker built with HTML, CSS, and JavaScript.</p>

<hr>

<h2>Overview</h2>
<p>
  BudgetBliss is a lightweight web app for managing income and expenses.
  It runs entirely in the browser using <code>localStorage</code> — no backend or setup required.
  This version is the MVP, focused on clean design, speed, and core functionality.
</p>

<hr>

<h2>Features</h2>

<h3>Core</h3>
<ul>
  <li>Add, view, and delete income and expense transactions</li>
  <li>See total balance, income, and expense stats</li>
  <li>View recent transactions with formatted dates and color indicators</li>
  <li>Data is stored locally using <code>localStorage</code></li>
</ul>

<h3>Management</h3>
<ul>
  <li>"Clear All Data" option with confirmation</li>
  <li>Storage info panel showing usage and transaction count</li>
  <li>Automatic storage space warnings</li>
</ul>

<h3>UI</h3>
<ul>
  <li>Minimal, modern layout</li>
  <li>Responsive for both desktop and mobile</li>
  <li>Gradient navbar and card-based dashboard</li>
</ul>

<hr>

<h2>Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> HTML, CSS, Vanilla JS</li>
  <li><b>Storage:</b> Browser localStorage</li>
  <li><b>Frameworks:</b> None</li>
  <li><b>Version:</b> MVP v1.0 (single-user, client-side only)</li>
</ul>

<hr>

  <li>Clone the repo:</li>
